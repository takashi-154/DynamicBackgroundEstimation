# Dynamic Background Estimation

某天体画像処理ソフトウェアのオマージュ的処理プログラム。
天体画像の背景カブリを推定し減算処理することで除去します。

# Requirement

* astropy 4.1
* matplotlib 3.3.2
* numpy 1.19.2
* scipy 1.5.2
* tifffile 2020.9.3

# Installation

```bash
pip install -r requirements.txt
```
